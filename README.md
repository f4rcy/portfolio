## ✨ Sobre el proyecto

Portfolio personal desarrollado desde cero con HTML, CSS y JavaScript puro. Diseño oscuro con estética moderna, animaciones suaves y efecto de brillo que sigue el cursor.

- ✨ Efecto de brillo que sigue el cursor
- ⌨️ Texto con efecto de escritura animada
